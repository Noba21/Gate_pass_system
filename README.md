# 🚪 Gate Pass System

## 📌 Overview
The Gate Pass System is a web-based application designed to manage and monitor the movement of visitors and employees within an organization. It helps automate gate pass issuance, track entry and exit records, and improve security and operational efficiency.

---

## ✨ Features

- 🔐 Visitor Management
  - Add and manage visitor details
  - Track entry and exit time

- 🧾 Gate Pass Generation
  - Create and manage gate passes
  - Assign purpose and destination

- 🏢 Employee Out-Duty (OD)
  - Employees can request gate passes
  - Track departure and return time

- 📊 Reports & Monitoring
  - View all records in real-time
  - Monitor active visitors and employees

- 📁 Data Export
  - Export records (Excel/CSV)

- 🎯 User-Friendly Interface
  - Simple and easy-to-use design

---

## 🛠️ Technologies Used

- Backend: ASP.NET Core / MVC  
- Frontend: HTML, CSS, JavaScript, Bootstrap  
- Database: SQL Server  
- ORM: Entity Framework Core
