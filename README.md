🚪 Gate Pass System
📌 Overview

The Gate Pass System is a web-based application designed to manage and monitor the movement of visitors and employees within an organization. It helps automate gate pass issuance, track entry/exit records, and improve overall security and efficiency.

✨ Features
🔐 Visitor Management
Record visitor details (name, company, purpose)
Track entry and exit times
🧾 Gate Pass Generation
Create and manage gate passes
Assign purpose and destination
🏢 Employee Out-Duty (OD)
Employees can request gate passes for official work
Track departure and return time
📊 Reports & Monitoring
View real-time records
Track active visitors and employees
📁 Data Export
Export records (Excel/CSV)
🎯 User-Friendly Interface
Easy navigation and simple design
🛠️ Technologies Used
Backend: ASP.NET Core / MVC
Frontend: HTML, CSS, JavaScript, Bootstrap
Database: SQL Server
ORM: Entity Framework Core
